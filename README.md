# Bank Churn Analysis (SQL Project)
SQL project analyzing banking customer behavior and credit risk using MS SQL Server

## Database Used
MS SQL Server

## Project Overview
This project analyzes bank customer data to uncover insights related to financial behavior, credit utilization, income patterns, and risk assessment.

## Key Focus Areas
- Customer Segmentation
- Credit Utilization & Risk Analysis
- Spending Behaviour & Financial Health
- High-Value Customer Identification
- Dormant & High-Risk Accounts

## Business Impact
- Better Credit Risk Management
- Personalized Banking Services
- Marketing Optimization
- Enhanced Customer Retention Strategies

## Skills Demonstrated
- Aggregations
- CASE Statements
- GROUP BY
- Risk Categorization
- Business KPI Analysis
